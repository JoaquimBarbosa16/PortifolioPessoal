Minhas Habilidades

HTML5

HTML é a linguagem de marcação usada para estruturar páginas da web. Ele define títulos, parágrafos, links, imagens e muito mais.

CSS3

CSS é a linguagem de estilos usada para definir o visual das páginas, como cores, fontes, tamanhos e layout responsivo.

JavaScript

JavaScript é uma linguagem de programação que permite criar interatividade nas páginas web, como animações, botões funcionais e formulários dinâmicos.

Java

Java é uma linguagem de programação robusta e amplamente usada para desenvolver aplicativos desktop, Android e sistemas back-end.

Git & GitHub

UI/UX Design
